# Screenshots

This directory contains screenshots for the IOC Scanner Lite documentation.

## How to Add Your Screenshots

Copy your screenshot files to this directory:

1. `screenshot-header.png` - Dashboard header and status  
2. `screenshot-results.png` - Scan results and analytics

### On Windows PowerShell:
```powershell
Copy-Item "C:\Users\rajsh\OneDrive\Pictures\Screenshots\Screenshot 2026-02-13 202739.png" "screenshot-header.png"
Copy-Item "C:\Users\rajsh\OneDrive\Pictures\Screenshots\Screenshot 2026-02-13 202700.png" "screenshot-results.png"
```

### On Linux/macOS:
```bash
cp ~/Pictures/screenshot-header.png ./screenshot-header.png
cp ~/Pictures/screenshot-results.png ./screenshot-results.png
```

## Image Naming Convention

- `screenshot-header.png` - Dashboard header with logo and status
- `screenshot-results.png` - Scan results panel with analytics
- `screenshot-*.png` - Other feature screenshots as needed

Once you've added the images, they will automatically appear in the README.md file.
